FELS
======

Front End Live Stream

FELS为您提供了一套由gulp驱动的动态开发环境，解决团队开发中规范化，模块化，自动化问题

## 启动web服务(dev)

```shell
node server.js
```

## 文件部署

```shell
gulp publish <options>
```

## 文件自动修复(beta)

```shell
gulp fix <options>
```
按照[CSS规范](./docs/style_standard.md)或者[JS规范](./docs/script_standard.md)自动修复源文件

## 安装

```shell
npm i
```

> 功能名称后的括号内，代表该功能目前开发状态
